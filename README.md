# tedbot

>   My discord bot

## Resources used

-   [Discord API](https://discordapp.com/developers/docs/intro)
-   [Node.js](https://nodejs.org/)

## Instructions

-   Add `auth.json` in the repository root, then add your discord bot token as the value for the `token` key
    ```sh
    {
      "token": "<YOUR_TOKEN>"
    }
    ```
-   Install dependencies
    ```sh
    npm install
    ```
